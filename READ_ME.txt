Para compilar o programa, basta digitar no terminal - dentro da pasta src:
    make

Para executar o programa, basta digitar no terminal - dentro da pasta src:
    ./City

Comandos de controle:
Seta para cima (^) - aumenta a altitude do drone (aumenta Z)
Seta para baixo (v) - diminui a altitude do drone até pousar (diminui Z)
Seta para direita (>) - move drone para direita (aumenta X)
Seta para esquerda (<) - move drone para esquerda (diminui X)
Tecla F - move drone para frente (aumenta Y)
Tecla T - move drone para trás (diminui Y)
Tecla R - coloca o drone de volta na posição inicial
Tecla S - inicia o movimento do drone
Mais (+) - aumenta o zoom (zoom in)
Menos (-) - diminui o zoom (zoom out)
